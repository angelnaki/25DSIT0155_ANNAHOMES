STEPS TO RUN THE PROJECT.
1. Install XAMPP
2. Start Apache and MySQL 
3. Open File explorer, open local disk(C:)open a folder titled Xampp, tap a folder titled htdocs then tap to open a folder with the project title annahomes, then save all project files inside that folder.
4. Import database using localhost/phpMyAdmin
5. Update database credentials in db_config.php
6. Access the website at: http://localhost/annahomes

DATABASE IMPORT INSTRUCTIONS
1. Open phpMyAdmin
2. Create a new database named homes_db
3. Click on Import tab
4. Browse and select the database.sql file
5. Click Go to import
